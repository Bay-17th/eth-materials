# 기여 가이드

이 저장소의 학습 자료 개선에 기여해주셔서 감사합니다.

## 이 저장소의 목적

`eth-materials`는 **학습 자료를 제공하는 읽기 전용 저장소**입니다.

- 자료 내용의 오류 수정
- 설명 보완 및 명확화
- 새로운 예제나 다이어그램 제안

## 숙제 제출은 여기가 아닙니다

숙제 제출은 [eth-homework](../eth-homework) 저장소에서 PR을 통해 진행합니다.

이 저장소에 숙제를 제출하지 마세요.

---

## 기여 방법

### 1. Issue 생성

자료 개선을 제안하려면 Issue를 생성하세요.

- [콘텐츠 개선 제안 템플릿](.github/ISSUE_TEMPLATE/content-issue.md) 사용
- 관련 주차와 파일을 명시
- 현재 내용과 제안 내용을 구체적으로 작성

### 2. 환영하는 기여

다음과 같은 기여를 환영합니다:

- **오타 수정:** 잘못된 철자, 문법 오류
- **설명 보완:** 이해하기 어려운 부분의 추가 설명
- **예제 추가:** 개념 이해를 돕는 코드 예제
- **다이어그램 제안:** 시각적 설명이 도움될 부분
- **링크 업데이트:** 깨진 링크나 더 좋은 참고 자료

### 3. 검토 과정

1. Issue 생성 및 제안 내용 작성
2. 관리자 검토 (보통 1-3일)
3. 승인 시 관리자가 직접 반영하거나 PR 요청
