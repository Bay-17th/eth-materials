# 6주차: Beacon Chain과 Finality (최종 프로젝트)

## 학습 목표

이번 주를 마치면 다음을 할 수 있습니다:

- Beacon Chain의 구조와 역할을 설명할 수 있습니다
- Slot, Epoch, Finality의 관계를 이해합니다
- 6주간 배운 내용을 통합한 dApp을 완성할 수 있습니다

## 예상 소요 시간

**8-10시간**

- 이론: 2시간
- 최종 프로젝트: 6-8시간

## 학습 내용

### 이론

| 자료 | 설명 |
|------|------|
| [슬라이드](theory/slides.md) | Marp 기반 프레젠테이션 (수업용) |
| [설명 문서](theory/explanation.md) | 한글 상세 설명 (복습용) |

**핵심 개념:**
- Beacon Chain 아키텍처
- Slot (12초) → Epoch (32 slots) → Finality (2 epochs)
- 검증자 위원회와 블록 제안
- 이더리움 로드맵 (Surge, Scourge, Verge, Purge, Splurge)

### 최종 프로젝트

| 자료 | 설명 |
|------|------|
| [최종 프로젝트 명세](dev/final-project.md) | 요구사항 및 체크리스트 |
| [프론트엔드 템플릿](../resources/frontend-template/) | Next.js + wagmi + RainbowKit |

**프로젝트 요구사항:**
- Solidity 스마트 컨트랙트 (Foundry)
- React 프론트엔드 (wagmi + RainbowKit)
- Sepolia 테스트넷 배포
- 6주간 배운 개념 통합
